# Optimal Transport Framework (ot-framework)

Welcome to the Optimal Transport Framework! This repository provides a core framework for optimal transport calculations.

If you are looking to run specific optimal transport tasks using this framework, we recommend checking out the `run_ot` repository, which builds on this framework and provides a user-friendly interface.

## Getting Started

To get started with optimal transport calculations, follow these steps:

1. Clone this repository:

    ```bash
    git clone https://gitlab.cern.ch/aml/optimal-transport/ot-framework.git
    cd ot-framework
    ```

2. For detailed instructions on running optimal transport tasks, please refer to the README.md in the `run_ot` repository:

    [run_ot README.md](https://gitlab.cern.ch/aml/optimal-transport/run_ot)
